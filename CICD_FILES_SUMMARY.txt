  1. Configure GitHub Secrets (5-10 minutes)
     → Read: .github/SECRETS_SETUP.md
     → Required: SSH_PRIVATE_KEY, SSH_USER, SERVER_HOST, 
                 SERVER_ENV_PROD, CLIENT_ENV_PROD

  2. Enable GitHub Actions (if not already)
     → Go to repository → Actions tab → Enable

  3. Set up Branch Protection (recommended)
     → Settings → Branches → Add rule for 'main'

  4. Push to GitHub
     → git add .github/ CI_CD*.md README.md
     → git commit -m "ci: add GitHub Actions CI/CD pipeline"
     → git push origin main

  5. Test the Pipeline
     → Create a test PR and watch workflows run!
